# 🌟 欢迎来到 **SL_qqban** 插件！

**SL_qqban** 是一款为 **SL 游戏服务器** 提供远程管理功能的插件，使用了 **OneBot 协议** 与 **QQ 机器人** 集成。它让管理员能够通过 QQ 群或私聊发送命令，轻松管理游戏服务器，进行封禁、解封、踢出玩家等操作。无论你在哪里，只需通过 QQ，即可快速有效地控制你的服务器！🎮💕

✨ **核心功能：**
- 通过 QQ 远程管理服务器。
- 支持封禁、解封、踢出玩家等常用操作。
- 仅授权的 QQ 号可以执行管理命令，保障安全性。
- 所有操作实时反馈，轻松掌控服务器状态。
- 现在修改查询功能（详情见释放页面）
🎯 **想要开始使用？**
